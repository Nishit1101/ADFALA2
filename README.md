#  Flutter REST API Integration App

##  Objective

This project demonstrates integration of a public REST API into a Flutter application. The app fetches JSON data, parses it into Dart objects, and displays it using a ListView.

##  API Used

https://jsonplaceholder.typicode.com/posts

##  Features

* Fetch data from REST API
* Parse JSON response
* Display data using ListView
* Error handling
* Loading indicator

## ️ Technologies Used

* Flutter
* Dart
* HTTP package

##  Output

The app displays a scrollable list of posts including:

* Title
* Body content


##  How to Run

1. Clone repository
2. Run `flutter pub get`
3. Run `flutter run`

##  Conclusion

This project helped understand API integration, JSON parsing, and UI rendering in Flutter.
